Add these files for the app to build without asset errors:

- `icon.png` (1024×1024) – app icon
- `splash-icon.png` – splash screen image
- `adaptive-icon.png` (1024×1024) – Android adaptive icon foreground

You can copy them from a new Expo app: `npx create-expo-app@latest temp --template blank` then copy from `temp/assets/` into here.
